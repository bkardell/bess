Apollo Client Framework :: Core JavaScript

Examples

All of the examples are executable code in the page. These pages are designed to be used by QA as a source for interactive testing.

Notes about the format of the page:

  • The source code will be placed in a yellow box showing the markup and the BESS rules
  • The example will be placed in a light blue box
  • Any markup that is added by BESS dynamically will be blue
  • Any markup that provides interaction will be red

Documentation

The pages under documentation provide descriptions of BESS modules and sample code. These pages are designed for UX designers who wish to use BESS.

APIs

The pages under APIs provide JavaDoc like documentation for programmable use of the framework. These pages are designed for engineers who wish to use the framework.
